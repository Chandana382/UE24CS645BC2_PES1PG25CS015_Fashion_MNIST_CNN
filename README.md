# UE24CS645BC2_-PES1PG25CS050-_Fashion_MNIST_CNN

## Project Overview
A Convolutional Neural Network (CNN) built from scratch using only NumPy
on the Fashion MNIST dataset — no PyTorch or TensorFlow used.

## Architecture
- Convolutional Layer (with ReLU activation)
- MaxPooling Layer
- Flatten Layer
- Fully Connected Layer (with Softmax)

## Dataset
Fashion MNIST — 70,000 grayscale 28x28 images across 10 clothing classes

## Results
- Training Accuracy: ~72.2% (after 2 epochs)
- Test Accuracy: 76.2% (381/500)

## How to Run
1. Open the `.ipynb` notebook in Google Colab
2. Run all cells in order (Steps 1–10)
3. Training takes approximately 3–5 minutes
